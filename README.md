# Xmesskitch
thème PluXml 

Thème sur 2 colonnes aux couleurs pétantes. 
Tailles des colonnes et couleurs facilement modifiable depuis les variables CSS en début de la feuille de styles <code>theme.css</code>.
<table><img src="https://github.com/gcyrillus/Xmesskitch/blob/main/Xmesskitch/preview.png?raw=true"></table>

* Un lien d'évitement inclus
* Données structurées générées (site,breadcrumbs,article, )
* quelques attributs pour l'accessibilité ( exemple logo de l'entête decrit avec le nom et description du titre sutué dans le pied)
* passe en une colonne à 1200px - valeur modifiable , à accorder de preference aux largeurs des deux colonnes.
